# **Framework ACE – Atlas • Calendar • Effort**

O ACE é um framework de organização baseado em três pilares:

1. **A – Atlas**  
    Onde a informação vive?  
    Qual é o repositório mestre dessa demanda?
    
2. **C – Calendar**  
    Quando isso acontece?  
    Qual é a data, prazo ou frequência?
    
3. **E – Effort**  
    Quanto esforço isso exige?  
    Quanto tempo, energia ou complexidade envolve?
    

Ele funciona tanto para **projetos grandes** quanto para **tarefas diárias**, e serve para criar ordem e previsibilidade no teu fluxo de trabalho.

---

# **1. A – ATLAS**

“**Onde isso mora?**”

O Atlas é o **mapa**. É o local onde você documenta, estrutura, planeja e organiza mentalmente o que precisa ser feito.

É onde você responde:

- O que é isso?
    
- Por que isso existe?
    
- O que precisa ser entregue?
    
- Quais informações preciso ter à mão?
    
- Quais contextos, arquivos, links, documentos pertencem a isso?
    

Exemplos de Atlas:

- Página no Notion para um projeto (ex: “Helios Voting – On Premise”)
    
- Pasta no OneDrive/SharePoint
    
- Documento no Word ou Google Docs
    
- Repositório no GitHub
    
- Página Confluence (para o tribunal)
    
- Planner no Microsoft 365
    
- Arquivo Markdown com todo o escopo
    

**Regra:**  
→ **Cada projeto/rotina precisa ter um Atlas.**  
Se não tem Atlas, vira bagunça, vira fragmento mental.

No Atlas devem existir:

- Objetivo
    
- Escopo
    
- Deliverables
    
- Passos
    
- Riscos
    
- Checklist
    
- Links úteis
    
- Histórico de decisões
    
- Cronograma macro
    

O Atlas é estável.  
Ele não muda toda semana.  
Ele é o teu _manual mestre_.

---

# **2. C – CALENDAR**

“**Quando isso acontece?**”

O Calendar contém apenas:

- deadlines,
    
- reuniões,
    
- datas que importam,
    
- lembretes,
    
- checkpoints,
    
- follow-ups,
    
- entregas com prazo fixo.
    

A pergunta aqui é:

→ **Em que dia específico algo precisa acontecer?**

O Calendar NÃO guarda tarefas sem data.  
Isso causa poluição e ansiedade.

**O Calendar é usado apenas para:**

- dia e horário de execução obrigatória
    
- entregas inadiáveis
    
- agendas
    
- milestones críticas
    
- lembretes que precisam aparecer no futuro
    

Exemplos:

- “Entrega do ETP AST – 14/11”
    
- “Reunião sobre Helios – 10h”
    
- “Revisar parecer LGPD – segunda, 15h”
    
- “Seguimento com fornecedor Snyk – 30/10”
    
- “Pagamento parcela 3 empréstimo – 4 de julho”
    

O Calendar não guarda:

- ideias
    
- projetos
    
- listas de tarefas
    
- backlogs
    
- rotinas opcionais
    

Tudo isso vive no **Atlas**, não no Calendar.

---

# **3. E – EFFORT**

“**Quanto esforço isso exige?**”

O Effort classifica toda demanda segundo:

- tempo estimado
    
- energia mental
    
- dificuldade
    
- quantidade de passos
    
- dependências
    

É a régua que te ajuda a encaixar cada tarefa na vida real.

O Effort pode ser classificado por:

### **Esforço por duração**

- **Low:** < 15 minutos
    
- **Medium:** 15–45 minutos
    
- **High:** 45–120 minutos
    
- **Deep Work:** > 2 horas
    

### **Esforço por energia mental**

- Baixa energia: tarefas mecânicas
    
- Média energia: exige foco moderado
    
- Alta energia: exige atenção profunda
    
- Estratégico: exige raciocínio crítico
    

### **Esforço por complexidade**

- Simples
    
- Moderado
    
- Complexo
    

---

# **Como usar o ACE no dia a dia**

Toda vez que chega uma demanda, você pergunta:

### **1. Atlas – Onde isso vive?**

Crio/atualizo a página base de organização.

### **2. Calendar – Quando isso acontece?**

Se tem data, vai para o calendário.  
Se não tem data, fica só no Atlas.

### **3. Effort – Quanto isso exige?**

Classifico para saber quando executar.

Depois disso, você monta o seu:

### **Daily Board (lista do dia)**

Você só puxa do Atlas para o dia:

- coisas com prazo (vieram do Calendar)
    
- coisas leves (Low Effort)
    
- coisas que você quer adiantar
    

Isso elimina:

- ansiedade
    
- tarefas soltas
    
- “não sei por onde começar”
    
- procrastinação
